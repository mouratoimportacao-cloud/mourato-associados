# AI Assistant Rules — Mourato & Associados

## Fluxo Obrigatório

1. `git pull origin main` — SEMPRE antes de começar
2. Ler arquivos antes de modificar
3. Criar tag de rollback antes de features grandes
4. Implementar e testar LOCAL (`npm run dev`)
5. Validar build (`npx next build`)
6. Commit + Push somente após aprovação do usuário

## Não Fazer

- NÃO rodar `npx prisma migrate` (ORM é fake, banco é JSONB)
- NÃO remover código sem permissão
- NÃO fazer push sem testar local
- NÃO assumir que tem a versão mais recente (sempre pull)
- NÃO expor credenciais em código
